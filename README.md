## Бот для сбора заказов с фриланс‑площадок

Бот ищет заказы по ключевым словам (создать сайт, телеграм‑бот и т.п.) на указанных вами страницах и присылает их вам в личный чат Telegram с кнопкой‑ссылкой на заказ.

### 1. Установка

1. Убедитесь, что установлен Python 3.10+.
2. В терминале перейдите в папку проекта:
   ```bash
   cd tg_bot_frelance
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

### 2. Настройка .env

1. Создайте файл `.env` в корне проекта на основе примера:
   - скопируйте `.env.example` → `.env`;
   - откройте `.env` и заполните:

   - **BOT_TOKEN** — токен вашего бота от `@BotFather`;
   - **OWNER_CHAT_ID** — ваш `chat_id`, чтобы бот знал, куда слать заказы;
   - **SCAN_INTERVAL_SECONDS** — как часто проверять сайты (в секундах), например `300` (5 минут).

2. Чтобы узнать `OWNER_CHAT_ID`, запустите бота (см. ниже) и в Telegram отправьте ему команду:
   - `/whoami`
   - бот ответит вашим `chat_id`, вставьте его в `.env`.

### 3. Запуск бота

В терминале из папки проекта:

```bash
python bot.py
```

Бот запустится на long‑polling и начнёт периодически проверять сайты.

### 4. Использование команд

Все команды работают только для владельца (`OWNER_CHAT_ID`).

- **/start** — краткая справка по командам и текущему интервалу проверки.
- **/addsite &lt;url&gt;** — добавить страницу для мониторинга, например:
  - `/addsite https://example.com/projects`
- **/removesite &lt;url&gt;** — удалить сайт из мониторинга.
- **/sites** — показать список всех добавленных сайтов.
- **/scan** — выполнить ручной поиск новых заказов прямо сейчас.
- **/whoami** — показать ваш `chat_id`.

Новые найденные заказы приходят сообщением с текстом и кнопкой:

- кнопка **«Открыть заказ»** ведёт напрямую на страницу заказа.

### 5. Как работает поиск заказов

- Бот скачивает HTML с каждой заданной страницы.
- Ищет ссылки (`&lt;a href=...&gt;`) и проверяет текст вокруг них на наличие ключевых слов:
  - по умолчанию: создать/сделать сайт, лендинг, интернет‑магазин, telegram/телеграм‑бот и т.д.
- Если ссылка ещё не отправлялась ранее — бот считает её новым заказом и шлёт вам в чат.
- Уже виденные ссылки сохраняются в файле:
  - `seen_links.json` — чтобы не присылать дубликаты.
- Список сайтов хранится в `sites.json`.

> Важно: парсер универсальный и работает по общему принципу. Для конкретных фриланс‑сайтов (freelancehunt, kwork, fl.ru и др.) при необходимости можно дополнительно дописать специализированные парсеры под их HTML‑структуру.

### 6. Изменение ключевых слов

Список ключевых слов находится в файле `config.py` в переменной `DEFAULT_KEYWORDS`.
Вы можете добавить свои фразы, по которым хотите искать заказы (например, конкретные технологии или типы работ).
